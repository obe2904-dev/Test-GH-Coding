## Stage 2: Individual Menu Analysis

### Expert Persona (NEW):
```
Du er gastronomisk konsulent med indsigt i kulinariske trends og 
menupositionering. Du analyserer food & beverage-udbud professionelt 
og markedsorienteret.
```

**What this enables:**
- ✅ Recognizes culinary styles (klassisk, moderne, fusion)
- ✅ Understands market trends (naturvin, specialty coffee, street food fusion)
- ✅ Identifies positioning (casual dining, bistro, fine dining)
- ✅ Professional, analytical perspective (not marketing)

### Analytical Framework (NEW):
```
FOKUS:
- Stilart: Klassisk, moderne, fusion? Traditionel vs. nysgerrig?
- Positionering: Hvilken madkultur/trend reflekterer menuen?
- Unikke elementer: Hvad skiller dette sted ud?

ILLUSTRER med konkrete eksempler i parentes.
```

**What this captures:**
- ✅ Culinary style and positioning
- ✅ Unique differentiators
- ✅ Examples that illustrate (not promote)

### Example Pattern (NEW):
```
[STYLE/CATEGORY] ([example1, example2, example3])
```

**Model:**
```
"Traditionel dansk madkultur (smørrebrød, pariserbøf, æblekage) 
kombineret med internationale caféretter (falafel, eggs benedict, 
chicken alfredo)"
```

**Why parentheses matter:**
- Without: "Vi har pariserbøf og eggs benedict" (menu listing)
- With: "Dansk (pariserbøf) + international (eggs benedict)" (analytical illustration)

## Quality Guards (Maintained):

### ✅ Still Forbidden:
- ❌ Subjective language: "lækker", "fantastisk", "hyggelig"
- ❌ Sales promotion: "Vi tilbyder den bedste Pariserbøf"
- ❌ Target audience: "Familier og par"
- ❌ Invention: Making up dishes or characteristics

### ✅ Now Allowed (Critical Change):
- ✅ Examples in parentheses to illustrate style
- ✅ Preparation methods: "in-house produktion", "hjemmelavede"
- ✅ Signature program names: "Faust Specials", "Det Grønne Valg"
- ✅ Style indicators: "klassisk", "moderne", "fusion"
- ✅ Trend observations: "specialty coffee", "naturvin-fokus"

```
"Vi tilbyder Pariserbøf, FAVORITTEN, Moules Mariniers og Æblekage"
```

**Problem:** Marketing, not analysis

### ✅ Analytical Illustration:

```
"Traditionel dansk madkultur (pariserbøf, smørrebrød, æblekage) 
kombineret med international café (falafel, eggs benedict)"
```

Run SQL query:
```sql
SELECT 
  service_period_name,
  ai_summary,
  
  -- Check for character-rich elements
  CASE 
    WHEN ai_summary ILIKE '%positionering%' 
      OR ai_summary ILIKE '%positioning%'
    THEN '✅ Has positioning'
    ELSE '❌ Missing positioning'
  END as positioning_check,
  
  CASE 
    WHEN ai_summary ILIKE '%(%)%'
    THEN '✅ Has examples in parentheses'
    ELSE '❌ Missing examples'
  END as examples_check,
  
  CASE 
    WHEN ai_summary ILIKE '%lækker%' 
      OR ai_summary ILIKE '%fantastisk%'
      OR ai_summary ILIKE '%hyggelig%'
    THEN '❌ SUBJECTIVE LANGUAGE'
    ELSE '✅ Objective'
  END as quality_check
  
FROM menu_results_v2
WHERE business_id = 'f4679fa9-3120-4a59-9506-d059b010c34a'
ORDER BY service_period_name;
```

## Key Learnings Applied

### 1. User Insight: Purpose Drives Design
> "This is for persona generation, not consumer display"

**Result:** Redesigned for character-rich analysis, not clean descriptions

### 2. User Example: The Gold Standard
> "Retterne kombinerer traditionel dansk madkultur (smørrebrød, pariserbøf) 
> med internationale caféretter (falafel, eggs benedict)"

**Result:** Example-in-parentheses pattern as core methodology

### 3. User Concern: Flattening Effect
> "My fear is that this will be flattened until it ends up in persona"

**Result:** Preserved character through analytical illustration

### 4. User Question: Expert Persona
> "Persona that knows all trends - is this critical?"

**Result:** "Gastronomisk konsulent med indsigt i kulinariske trends" - absolutely critical
