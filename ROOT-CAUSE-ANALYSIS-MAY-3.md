## 🎓 LESSONS LEARNED

### Architecture Misconceptions
1. **Multi-phase generation is complex** - I modified Phase 2b thinking it controlled output
2. **Data flow is non-obvious** - strategicBrief killed between phases
3. **Fields have specific purposes** - `holidayContext` is THE mechanism, not a nice-to-have
4. **Functions are independent** - get-weekly-strategy ≠ generate-text-from-idea

### Development Process Failures
1. **Didn't check logs first** - Would have shown detection not firing
2. **Didn't trace full flow** - Missed the Phase A/Phase B split
3. **Didn't verify data** - Assumed event format without checking
4. **Deployed untested code** - No validation that changes work

### Prompt Engineering Limitations
1. **Override blocks aren't enforcement** - AI can ignore them
2. **Hierarchy matters** - Slot templates still override strategic directives
3. **Multiple prompts = multiple points of failure** - Each phase needs its own fix
